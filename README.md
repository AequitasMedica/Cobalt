# COBALT

## Bienvenida

En Aequitas Medica creemos que la credibilidad de un resultado es proporcional a la transparencia con la que fue construido.

Por eso, utilizamos herramientas de código abierto y compartimos nuestros scripts en repositorios públicos, con el propósito de que puedan ser revisados, auditados, adaptados y reutilizados por otros equipos académicos y científicos.

Después de implementar varios consensos tipo RAND/UCLA, hemos tenido la oportunidad de iterar y mejorar nuestra propia herramienta para facilitar este tipo de estudios. Hoy nos sentimos listos para compartir **COBALT** con cualquier equipo que desee realizar un consenso estructurado, transparente y reproducible.

**COBALT** está basada en la metodología descrita por Kathryn Fitch y colaboradores para la RAND Corporation. Es una aplicación web sencilla, desarrollada en HTML, JavaScript y PHP, por lo que puede ejecutarse en prácticamente cualquier dispositivo, ya sea en un servidor en línea o en un entorno local.

Su objetivo es facilitar la captura de respuestas, el seguimiento por rondas, el análisis de resultados, el cálculo de medianas, la evaluación del índice de desacuerdo y la visualización de los consensos alcanzados.

Para los equipos que no cuenten con la infraestructura técnica necesaria, Aequitas Medica pone a disposición su servidor, así como asesoría técnica, metodológica y científica para acompañar el proceso.

Este repositorio se comparte con la intención de promover procesos de consenso más transparentes, accesibles y reproducibles.

## Welcome

At Aequitas Medica, we believe that the credibility of a result is directly linked to the transparency of the process through which it was built.

For this reason, we use open-source tools and share our scripts in public repositories so they can be reviewed, audited, adapted, and reused by other academic and scientific teams.

After implementing several RAND/UCLA appropriateness method consensus processes, we have had the opportunity to iterate and improve our own tool to support this type of study. We are now ready to share **COBALT** with any team interested in conducting a structured, transparent, and reproducible consensus process.

**COBALT** is based on the methodology described by Kathryn Fitch and colleagues for the RAND Corporation. It is a simple web application built with HTML, JavaScript, and PHP, which allows it to run on virtually any device, either on an online server or in a local environment.

Its purpose is to facilitate response capture, round-by-round follow-up, result analysis, median calculation, disagreement index assessment, and visualization of consensus results.

For teams that do not have the necessary technical infrastructure, Aequitas Medica can provide access to its server, as well as technical, methodological, and scientific support throughout the process.

This repository is shared with the aim of promoting more transparent, accessible, and reproducible consensus processes.
